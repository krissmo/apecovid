# Impacto del COVID-19 en la Sociedad

Sitio web estático que analiza el impacto de la pandemia en Ecuador y el mundo, con secciones de introducción, noticias y estadísticas de mortalidad.

## Estructura de Archivos
- **index.html**: Página principal con introducción y video.
- **noticia.html**: Noticias recientes sobre COVID-19.
- **mortandad.html**: Tabla de estadísticas de mortalidad.
- **css/style.css**: Estilos globales.
- **img/**: Imágenes (logo, propaganda).
- **video/**: Videos educativos.

## Cómo Ejecutar Localmente
1. Abre `index.html` en un navegador
2. Navega por los enlaces en el menú.


## Dependencias
Ninguna (puro HTML/CSS). Compatible con navegadores modernos.
